Hexo 本地目錄遷移

發表於 05-28-2022 分類於 建站 ， Hexo

一句話：複製粘貼到目的地

緣起：剛學會配置 Hexo 網站時，創建在 macOS 用戶的根目錄，無法備份，不放心。又想著搬家到 Dropbox 才是及妥當的，於是開幹

  • 複製整個目錄到 Dropbox 根目錄（當時103M就有14950個文件，同步好久）
  • Terminal 中進入新目錄，一鍵三聯 完美本地運行
  • 測試一段時間，沒問題後，刪除之前到網站資料夾即可

繼續完善 Hexo 配置

一、在線編輯插件 Hexo Admin

  1. 安裝
    1. 在你的hexo站点目录下，安装 npm install --save hexo-admin
    2. 啟動本地服務 hexo s
    3. 進入網址在線編輯 http://localhost:4000/admin
  2. 創建密碼登錄 Authentification Setup Admin Config Section
    Copy this into your _config.yml, and restart Hexo. Now you’ll be protected with a password!

二、在 nexT 主題中配置 tags

  1. 在hexo根目錄的source文件夾下新建一個 tags 文件夾
    $ hexo new page "tags"
  2. 編輯 tags 文件夾中的 index.md 文件

免费图书下载站

發表於 05-26-2022 更新於 05-28-2022 分類於 資源分享 ， 電子書

引言

论全球最好的免费图书下载站，我觉得“Zli.brary”当之为全球最良心的站点，其资源齐全、种类繁多，还提供免费下载，深受全球网友的喜爱！

不过这种资源，当然会触碰到某些人的利益，所以经常域名会有打不开的情况。但这并没有阻止爱读书看书的小伙伴的热情。

这不，虽然打不开了，但新的域名“春风吹又生”，来，给大家介绍介绍这款全球最受欢迎的站点。

网站介绍

https://Zli.brary 的资源非常丰富，看看这个站点上收集的书籍已经有一千多万本了，我记得最开始安利的时候，才八百多万本书籍，这一年多的时候已经新增了快两百多万本了，只能说“牛啊”！

网站支持多国语言，共支持28种语言，不过默认的情况下，大家打开时，网站就会自动切换你所在地的语言了

在这里可以“通用搜索”和“全文搜索”。

非常出名的书籍当然是能搜索出来了，但有些网文小说，也能查出来，像下面的这部小说，能搜索出500条搜索源，也是不错的！

支持免费下载，不过下载的时候需要登录。而注册来说，其实就是邮件注册，无需手机号码注册，非常方便
