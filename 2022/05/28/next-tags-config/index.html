Hexo 主題 nexT 配置 tags

發表於 05-28-2022 分類於 建站 ， Hexo

繼續完善 Hexo 配置

一、在線編輯插件 Hexo Admin

  1. 安裝
    1. 在你的hexo站点目录下，安装 npm install --save hexo-admin
    2. 啟動本地服務 hexo s
    3. 進入網址在線編輯 http://localhost:4000/admin
  2. 創建密碼登錄 Authentification Setup Admin Config Section
    Copy this into your _config.yml, and restart Hexo. Now you’ll be protected with a password!

二、在 nexT 主題中配置 tags

  1. 在hexo根目錄的source文件夾下新建一個 tags 文件夾
    $ hexo new page "tags"
  2. 編輯 tags 文件夾中的 index.md 文件
