用mac錄影錄音

Running Man
06-09-2022
Apple › macOS
影音錄製

https://youtu.be/2WMD62FszGA

一、安裝 Soundflower

  1. 項目地址：https://github.com/mattingalls/Soundflower/releases/tag/2.0b2
  2. 下載 Soundflower-2.0b2.dmg（已備份到阿里雲id134）
  3. 安裝前先將此項中的🔒點開 System Preferences - Security & Privacy - Click the lock to make changes.
  4. 安裝完成，記得再點🔒，確保鎖定狀態
  5. 重啓 macOS

二、配置

  1. 檢查裝置 System Prefrences - Sound - Output / Input 這兩項中均出現了兩個項目：Soundflower (2ch / 64ch)
  2. 啟動 音訊MIDI設定：Launchpad - Ohter - Audio MIDI Setup
  3. 點左下角加號，Creat Aggregate Device（製作聚集裝置），Audio Device 只選這兩項
    1. MacBook Pro Microphone（內建麥克風），打勾後面的 Drift Correction (漂移校正，默認未選中），可保证语音输入法和錄音功能正常使用。注意：可能會導致環境音干擾，根據實際情況取捨此項
    2. Soundflower (64ch) ，默認 Drift Correction 打勾在此項
  4. 重複上步驟，Creat Multi-Output Device（製作多重輸出裝置），同樣選中兩項，默認 Drift Correction 打勾在Soundflower (64ch)
  5. 日後不用，可在 Audio Devices 中刪除這兩個創建的項目
  6. 再次檢查裝置 System Prefrences - Sound - Output / Input 這兩項中均 多出了幾項
    1. Output 增加两项: Aggregate Device Multi-Output Device
    2. Input 增加一项: Aggregate Device

三、錄製系統內置音頻

  1. 啟動 QuickTime Player，File - New Audio Recoding
  2. 點小紅點右側按鈕，選中 Soundflower (64ch)
  3. enjoy!
